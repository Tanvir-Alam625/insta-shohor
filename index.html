2. What does javascript event loop?

Answer: The relationship between the call back queue and the call stack is called the event loop.

a constantly running process that coordinates the tasks between the call stack and callback queue to achieve concurrency.

JavaScript is used in asynchronous programming event loops. Event Loop stores JavaScript's asynchronous programming code in the browser's Web API, from where it comes to the callback queue, and when the call stack is free, JavaScript sends that code to the call stack via Event Loop.
